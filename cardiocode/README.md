# CardioCode Package

This is the main Python package for CardioCode.

## Structure

```
cardiocode/
  mcp/           - MCP server and tools
  ingestion/     - PDF processing and knowledge extraction
  guidelines/    - Clinical guideline implementations
```

## Usage

Run the MCP server:
```bash
python -m cardiocode.mcp.server
```

See the main [README](../README.md) and [INSTALLATION.md](../INSTALLATION.md) for full setup instructions.
